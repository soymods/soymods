What is soymods?

Minecraft ideas, shipped

The site is a central place for projects that would otherwise be scattered across downloads, source repositories, videos, and release pages.

Built by ryduzz

I use soymods as the home base for Minecraft work I want to share publicly. Some projects are polished mods, some are utilities for recording or playing, some are plugins or experiments, and some are smaller ideas that become useful enough to release.

The goal is to keep everything easy to find: what the project does, where to download it, where the source code lives, what versions it targets, and where to follow updates. As the project list grows, this page and the project pages act as the index for the broader soymods ecosystem.

Project notes soymods projects usually start as things I want in my own Minecraft setup, then become public when they are useful enough for other players or server owners.
